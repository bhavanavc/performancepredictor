# performancepredictor


Student Performance Predictor (BYOP Project)
 Overview
This project is developed as part of the BYOP activity for Fundamentals of AI & ML. It is a simple machine learning application that predicts a student’s academic performance based on daily habits. The project demonstrates how AI can be used to analyse data and make meaningful predictions.

Problem Statement
Students often do not realize how factors like study time, attendance, and sleep affect their performance. There is a need for a simple system that helps in understanding these relationships.

 Solution Approach
The project uses a Decision Tree Classifier, a supervised machine learning algorithm. The model is trained on a dataset containing student details and their performance levels.

Input Features:
•	Study Hours
•	Attendance (%)
•	Sleep Hours

Output:
•	Performance Prediction (Good / Average / Poor)
The model learns patterns from the data and predicts the performance based on user input.

 Technologies Used
•	Python
•	Pandas (for data handling)
•	Scikit-learn (for machine learning model)
•	Visual Studio Code
 Working of the Project
The dataset is first loaded using Pandas. The features (study hours, attendance, sleep) are separated from the target (performance). The model is trained using this data, and when the user enters values, the trained model predicts the performance based on learned patterns.
 Learning Outcomes
•	Understanding of basic machine learning concepts
•	Experience with data preprocessing and model training
•	Importance of data in making predictions
•	Problem-solving and debugging skills
